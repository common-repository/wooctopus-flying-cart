== Description ==

Flying Product to Mini Cart for WooCommerce allows adding beautiful flying animation when the customer adding a product to the cart. This gives a dynamic and improves the usability of your online store. Pay attention: this version of the plugin works only with the Elementor PRO mini cart. 

== Installation ==

1. Unzip the downloaded zip file and upload the plugin folder into the `wp-content/plugins/` directory. Alternatively, upload from Plugins >> Add New >> Upload Plugin.
2. Activate the plugin through the 'Plugins' menu in WordPress
